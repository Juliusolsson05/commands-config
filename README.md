# commands-config

Shared AI agent commands for use with [agent-mgr](https://github.com/Juliusolsson05/agent-manager).

## Commands

- **simple-code** — Enforce simple, boring, obvious code over clever abstractions
- **no-overengineering** — Stop AI from inventing frameworks when stdlib is enough

## Usage

```bash
amgr sync-repo add Juliusolsson05/commands-config
amgr sync
```
